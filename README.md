# Taxi Revenue Maximization Analysis

## Overview
This project analyzes taxi trip data to understand how payment methods influence fare amounts and overall revenue, with the objective of identifying data-driven strategies to maximize driver earnings.

## Key Analysis
- Exploratory data analysis and data cleaning
- Comparison of fare distributions across payment types
- Statistical hypothesis testing and regression analysis
- Visualization of revenue and payment trends

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- SciPy
- Jupyter Notebook

## Outcome
The analysis highlights significant differences in fares across payment methods and provides actionable insights to support revenue optimization decisions.
